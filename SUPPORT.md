Voici quelques erreurs qui peuvent se produire et comment résoudre le problème :

# Version 8.1.14 installable
## Erreur :
`TypeError : unhashable type: 'list'`

## Remède :
Supprimer le fichier `C:\ProgramData\pySequence\sequence.cfg`
